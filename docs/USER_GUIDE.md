# 比价智能体 - 用户使用指南

## 🆕 新功能使用指南

### 1. **查看 AI 思考过程** 🧠

当 AI 完成商品比价后，你会看到：

```
💡 最低价推荐... (推荐语)

[🧠 思考过程 (XX 字)] ← 点击这里展开
└─ 详细的分析步骤...
   - 理解您的需求
   - 筛选符合条件的商品
   - 匹配度分析
   - 生成最终推荐

📊 价格对比分析
└─ [导出] ← 新增：可以导出图表！
```

**使用方式**:
- ✅ 点击 `[🧠 思考过程]` 展开，了解 AI 如何分析的
- ✅ 再点击一次可以收起，节省屏幕空间
- ✅ 对比多个结果时非常有用

---

### 2. **查看商品完整信息** 📋

表格中每行商品右侧有"查看详情"按钮：

```
┌─────────────────────────────────────┐
│ iPhone 15 Pro (128GB)               │
│ 平台: [京东]  价格: ¥7999           │
│ 规格: 6.1寸 A17 Pro 120Hz...        │
│ 匹配度: [完全匹配]  [查看详情] ← ← │
└─────────────────────────────────────┘
```

**详情抽屉内容** 包括：
- 📝 完整商品名称（无截断）
- 💰 当前价格 + 原价（如有）
- ⭐ 评分 + 评价数
- 📦 完整规格参数
- ✨ 主要特性标签
- 🚚 运费信息
- 🔗 "查看商品详情" 按钮

**快速使用**:
```
1. 找到感兴趣的商品
2. 点击 "查看详情" 按钮
3. 在弹出的抽屉中查看完整信息
4. 点击 "查看商品详情" 跳转到商城
5. 点击右上角 ✕ 关闭抽屉
```

---

### 3. **表格内容全显** 🔍

**改进点**:

| 之前 | 现在 |
|-----|-----|
| 名称被截断: iPhone 15 Pro Max... | 悬停显示完整: iPhone 15 Pro Max (512GB 深空黑) |

### 4. **图表导出功能** 📊

价格对比图表右上角新增"导出"按钮：

```
📊 价格对比分析      [导出] ← 点击导出 PNG
┌─────────────────┐
│    条形图表      │
│ 京东 淘宝        │
└─────────────────┘
```

**导出使用**:
1. 点击右上角 `[导出]` 按钮
2. 浏览器自动下载 PNG 图片
3. 文件名: `price-comparison-{时间戳}.png`
4. 可用于分享、报告、演示

**导出优势**:
- 高质量 2x 分辨率 (2400x 高度)
- 深色主题保持
- 包含所有数据信息

## 📱 响应式布局说明

### 设备适配

**桌面端** (1920px+)
- 全功能显示
- 表格完整展示
- 图表大尺寸

**平板端** (768px - 1919px)
- 布局自动缩放

- 竖向单列布局
- 气泡宽度 75% + 边距
- 详情改为抽屉显示（全屏）
- 图表高度自适应

### 缩放支持

```
浏览器缩放 (Ctrl + 或 Cmd + ±)

100% (标准)    ← 推荐
125% (中等)    ← 舒适
150% (大)      ← 适合视力困难
200% (超大)    ← 可用但需要更多滚动
```

## 🆘 常见问题

### Q: 思考内容不显示？
**A**: 
- 检查是否已完成搜索（加载圈消失）
- 尝试刷新页面 (F5)
- 检查网络连接

### Q: 表格内容还是显示不全？
**A**:
- 先尝试鼠标悬停查看 Tooltip
- 点击"查看详情"查看完整信息
- 如果还是不行，尝试缩放页面 (Ctrl + -)

### Q: 导出图表失败？
**A**:
- 检查浏览器是否允许下载
- 尝试在不同浏览器中操作
- 清除浏览器缓存后重试

### Q: 移动端显示混乱？
**A**:
- 确保浏览器宽度 ≥ 320px
- 尝试旋转屏幕（如果支持）
- 清除缓存重新加载

## 📞 技术支持

如果遇到问题，请:

1. **检查浏览器**
   - Chrome 90+ / Safari 14+ / Firefox 88+ 建议
   - 其他浏览器可能显示不完整

2. **清除缓存**
   - Ctrl + Shift + Delete (浏览器缓存)
   - 然后 F5 刷新

3. **检查控制台**
   - F12 打开开发者工具
   - Console 标签查看是否有红色错误
   - 将错误信息保存用于反馈

4. **提交反馈**
   - 描述具体操作步骤
   - 附上浏览器和设备信息
   - 最好附上截图
